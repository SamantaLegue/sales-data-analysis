# 📊 Sales Data Analysis Project

## 📌 Project Overview
This project analyzes sales data to identify trends, patterns, and key business insights. The goal is to support better decision-making using data.

---

## 📁 Dataset
- File: sales report.xlsx
- Contains sales transactions, revenue, and performance data.

---

## 🎯 Objectives
- Analyze total sales performance  
- Identify top-performing products  
- Understand sales trends over time  
- Detect patterns that can improve business decisions  

---

## 🛠 Tools Used
- Microsoft Excel  
- Data Cleaning  
- Data Analysis  
- Data Visualization  

---

## 📊 Key Insights (UPDATE AFTER ANALYSIS)
- Top-selling products  
- Monthly revenue trends  
- Customer or regional performance
